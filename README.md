### 🚗 AutoCare-AI
Mecânico virtual em Flutter com gestão de documentos + OCR, pipeline RAG (pgvector + Supabase), chatbot multi-modo, deteção visual em tempo real com YOLO e integração com a NHTSA vPIC API e Gemini AI.

### 🎥 NeRF Studio
Pipeline de reconstrução 3D construído de raiz com Python, PyTorch, FastAPI, React, Instant-NGP e COLMAP — correndo numa RTX 3070 Ti.

### 🏆 MatchUp
App Android de gestão de torneios desportivos, com frontend em Kotlin/Compose e backend em Ktor, deployado no Render com Supabase PostgreSQL.
